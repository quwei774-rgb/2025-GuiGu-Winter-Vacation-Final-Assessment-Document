## 25届寒假终期结果提交说明

### 一、注意事项

1. 以压缩包的形式打包上传

2. 需重命名为 【专业班级姓名】，放至“提交至此文件夹内”
-----------

1. 在仓库首页点击fork（复刻），创建仓库副本至自己的账号下
